# mikrotik

MikroTik RouterOS device management — a first-party [orca](https://github.com/argyle-labs/orca) **service
backend** plugin. Exposes no tools of its own; orca drives it through the generic
`service.*` surface (deploy, backup, restore, configure, status, connect, sync).

Modalities: **device**. Status: **scaffolded** — see
[CAPABILITIES.md](CAPABILITIES.md). Generated by `orca/scripts/scaffold-plugin.sh`.

## Manual setup & management

This plugin automates setup, but the repo is self-contained: the docs below (migrated + anonymized from the homelab runbook) let you deploy, configure, and operate mikrotik **entirely by hand** across every supported runtime. The plugin just makes it easier.

- [mikrotik-setup](docs/mikrotik-setup.md)
